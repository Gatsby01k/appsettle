API-ready settlement infrastructure for payment operators.

Build settlement creation, treasury quotes, status tracking, webhooks, and reconciliation exports into your operational workflows.

Core endpoints

  • POST /v1/settlements create a settlement
  • GET /v1/treasury/quote request corridor quote
  • GET /v1/settlements/{id} track state
  • POST /v1/webhooks configure event delivery
